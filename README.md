# 🐶🐱 Dog vs Cat Image Classification

## 📌 Project Overview
This project was developed using Google Teachable Machine to classify images into two categories: Dog and Cat. The trained model was exported in TensorFlow Keras format and integrated with a Python script that predicts the class of an input image.

---

## 📂 Classes
- 🐶 Dog
- 🐱 Cat

---

## 🛠️ Tools and Technologies
- Google Teachable Machine
- Python
- TensorFlow / Keras
- NumPy
- Pillow
- Google Colab / VS Code

---

## 🚀 Project Workflow

1. Created an Image Project in Google Teachable Machine.
2. Added two classes: Dog and Cat.
3. Uploaded training images for each class.
4. Trained the image classification model.
5. Exported the model in TensorFlow (Keras) format.
6. Developed a Python script to load the trained model.
7. Tested the model using an input image.
8. Displayed the predicted class and confidence score.

---

## 📁 Project Files

- keras_model.h5 – Trained Keras model.
- labels.txt – Class labels.
- untitled3.py – Python prediction script.
- README.md – Project documentation.

---

## ▶️ How to Run

1. Install the required Python libraries.
2. Place the input image in the project folder.
3. Run the Python script.
4. The program will display the predicted class and its confidence score.

---

## ✅ Output

The model successfully predicts whether the input image is a Dog or a Cat, along with the confidence percentage for the prediction.
